## 📌 Overview

An interactive data dashboard for exploring clusters of cargo ships based on their operational and physical characteristics. Applies unsupervised learning (clustering) to maritime data, with interactive visualisations for insight discovery.

**Tech Stack:** `Python` `Clustering` `Data Visualisation` `Plotly` `Maritime Data`

---

## 🚀 Getting Started

### Prerequisites

```bash
pip install -r requirements.txt
```

> Requires **Python 3.8+**

### Installation

```bash
git clone https://github.com/Onabanjomicheal/Cargo-Ship-Dashboard.git
cd Cargo-Ship-Dashboard
pip install -r requirements.txt
```

### Run

```bash
python main.py
```
